## How to run

Install [dotnet SDK](https://dotnet.microsoft.com/download) and run locally or use [![gitpod](https://img.shields.io/badge/Open%20in-Gitpod-C8597A.svg)](https://gitpod.io/#https://github.com/jcaillon/AdventOfCode2021).

```bash
dotnet run
```
